# Basirion Design System Documentation

Complete documentation for the Basirion Design System.

## Overview

The Basirion Design System is a modern, vibrant design system combining:
- **Professional Neutral Palette**: Inspired by Linear/Vercel
- **Fiery Theme**: Custom vibrant colors (Ember Red, Flame Orange, Sunset Gold, etc.)
- **Modern UI Patterns**: Glassmorphism, smooth animations, responsive design

## Color Palette

### Neutral Colors
- `basirion-white` through `basirion-gray-900`
- Professional grays for backgrounds and text

### Fiery Theme Colors
- `fire-ember` - Ember Red (#FF3B30)
- `fire-flame` - Flame Orange (#FF9500)
- `fire-gold` - Sunset Gold (#FFCC00)
- `fire-magenta` - Deep Magenta (#FF2D55)
- `fire-purple` - Volcanic Purple (#AF52DE)

## Utility Classes

### Cards
- `.basirion-card` - Standard card with hover effects

### Buttons
- `.basirion-button-primary` - Primary button with fiery gradient

### Inputs
- `.basirion-input` - Standardized input styling
- `.basirion-select` - Standardized select styling

### Links
- `.basirion-link` - Standardized link styling

### Badges
- `.basirion-badge` - Base badge
- `.basirion-badge-primary` - Primary variant
- `.basirion-badge-success` - Success variant
- `.basirion-badge-warning` - Warning variant
- `.basirion-badge-error` - Error variant

### Responsive Utilities
- `.text-responsive` - Responsive text sizing
- `.p-responsive` - Responsive padding
- `.gap-responsive` - Responsive gaps
